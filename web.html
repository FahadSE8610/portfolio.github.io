About Me

My name is Emily Trabert. I'm currently a Software Development Engineer and Computer Science Tutor in Durham, NC. I graduated from American University in Washington, DC with a BS in Computer Science, a BS in Audio Technology, and a Minor in Graphic Design.

I tailor my teaching style to each individual student based on how each one learns best. I also like to learn what each student is passionate about and make analogies about computer science to their passion. I believe learning should be about discovering practical skills for use in day-to-day life, not memorization.

I tutor a broad range of computer science subjects including basic programming concepts as well as the following languages:

  • Java
  • Python
  • C#
  • SQL
  • HTML/CSS/JavaScript
